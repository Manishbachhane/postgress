----->MVCC
PostgreSQL me MVCC ka full form hota hai Multi Version Concurrency Control.
Ye PostgreSQL ka system hai jo multiple users ko same time pe database use karne deta hai
without heavy conflicts. Simple words me MVCC database ke data ki multiple versions maintain
karta hai taki ek user ka operation dusre user ko unnecessarily block na kare.

Example ke liye maan lo ek user kisi row ko update kar raha hai aur usi time dusra user same
table ko read kar raha hai. Without MVCC database ko pura row/table lock karna padta aur dusre
users wait karte. Lekin PostgreSQL MVCC use karta hai, isliye update hone ke baad bhi dusra user
purani safe version read kar sakta hai jab tak transaction complete nahi hoti.
